# Django Polls App

A simple polling web application built while following the [official Django tutorial](https://docs.djangoproject.com/en/stable/intro/tutorial01/).

## Tech Stack

- Python
- Django

## Setup

```bash
# Clone the repo
git clone https://github.com/your-username/django-polls.git
cd django-polls

# Create and activate virtual environment
python -m venv venv
venv\Scripts\activate  # Windows
source venv/bin/activate  # Mac/Linux

# Install dependencies
pip install -r requirements.txt

# Run migrations
python manage.py migrate

# Start the server
python manage.py runserver
```
